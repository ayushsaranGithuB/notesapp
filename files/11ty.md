# Personal Blog
##  A NetlifyCMS powered static blog site generated with 11ty


A JAM-Stack Static Blog generated server-side using *11ty.js* with content stored in git and accessed via the *DecapCMS*



### TECH STACK:

- **Backend:** Node.js
- **UI:** Tailwind
- **SSG:** 11ty.js
- **Hosting & CD:** Netlify
