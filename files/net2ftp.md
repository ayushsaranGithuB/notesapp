# net2ftp
##  a web based FTP and SSH client.

### Features
- Navigate the FTP server | Upload and download files
- View and Edit code with syntax highlighting
- Copy, move, delete to 2nd FTP server
- Zip and unzip files

[net2ftp](https://net2ftp.alwaysdata.com/)
